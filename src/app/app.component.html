Building Blocks to modern Angular development

  • Modules
    • Perform types of tasks and exports values like a class
  • Libraries
    • They have the @angular prefix.
  • Components
    • They are basiclly Angular classes
  • Templates
    • The HTML view
  • Data Binding
    • Both one way and two way data binding
  • DirectiveS
    • For DOM manipulation
  • Services
    • Classes with functions
  • Dependency Injections
    • Constructors that request services.

Life Cycle Hooks

  • ngOnChanges()
    • This hook is Called before ngOnInit() and whenever one or more data-bound input properties change.()
  • ngOnInit()
    • This hook is only called once after the first ngOnChanges. it sets the components or directives input properties
  • ngOnDestroy()()
    • Detaches Observables and event handlers. This hook is called before Angular does its on destroy
